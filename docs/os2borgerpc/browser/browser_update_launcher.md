## Beskrivelse
Dette script udskifter browseren (der som standard er Firefox) i venstremenuen med en valgfri browser, angivet som første parameter.

Valgmulighederne er:
- firefox
- google-chrome
- microsoft-edge

Dette script er blevet testet og virker på Ubuntu 22.04.
